# Лабораторная работа №10 - Словарь
В этой лабораторной работе была разработана программа, которая является словарем стран и столиц.
# Cреда разработки
Язык программирования: Python.  
Среда разработки: Visual Studio 2019.
# Словарь Страны и столицы
Необходимо разработать программу, в которой используется словарь с названиями ряда государств и их столицами (не менее 10 - в текстовом файле или в коде).  
Программа должна обеспечивать:  
- вывод столицы заданного государства;  
- вывод государства, столицей которого является заданный город.

Если заданного государства или города в словаре нет, на экран должно выводиться соответствующее сообщение.  
Требуется:  
1. Написать на любом языке программу, реализующую поставленную задачу.  
- Входные данные: название столицы или государства.  
- Выходные данные: название государства или столицы соответственно.  
2. Обеспечить достаточный уровень качества ПО: 
Код программы должен содержать комментарии для созданных функций и основных блоков, качественный диалог с пользователем и обработку ошибок ввода.
# Инструкция по работе
Запустите файл программы с названием main.py. После запуска появится меню с четырьмя вариантами действий. Для поиска столицы страны выберите пункт 1 и введите название государства. Для поиска страны по столице выберите пункт 2 и введите название города. Чтобы увидеть все страны из базы данных, выберите пункт 3. Для выхода из программы выберите пункт 0.  
Программа работает с 12 странами: Россия, Германия, Франция, Италия, Испания, Великобритания, Япония, Китай, США, Канада, Бразилия, Австралия. Названия можно вводить в любом регистре - программа сама исправит написание. Если введенной страны или столицы нет в базе, вы получите сообщение об ошибке.
